Como iniciar o projeto:

1. inicar Ollama local:
- em um terminal novo, rode "ollama start"
- em um terminal novo, rode "ollama pull llama 3"

2. inicar o backend
- rode GeradorCurriculoApplication pelo IDE

3. inicar o frontend
- dentro da pasta frontend, rode "npm run dev"
- abra o projeto no localhost indicado no terminal
